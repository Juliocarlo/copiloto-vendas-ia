# Guia dos Temas - Copiloto Premium

Este guia resume cada arquivo da pasta `temas` com o que cobre e as perguntas principais.  
Assim o vendedor encontra rapidamente o assunto certo durante o atendimento.

| Tema                  | O que cobre                                      | Perguntas principais |
|-----------------------|--------------------------------------------------|----------------------|
| Notebooks             | Tipos, upgrades, mobilidade                      | Diferença básico/gamer, SSD, RAM |
| Desktops              | Tipos, montagem personalizada, desempenho        | Diferença notebook/desktop, PCs prontos |
| Processadores         | Intel e AMD, compatibilidade                     | Qual CPU para jogos, multitarefas |
| Memória               | DDR3, DDR4, DDR5, capacidade                     | 8GB vs 16GB, upgrade notebook |
| Armazenamento         | HD, SSD, NVMe                                    | Diferença HD/SSD, usar juntos |
| Placas de Vídeo       | Integradas vs dedicadas                          | Preciso de GPU, melhor para jogos |
| Placas de Rede        | Ethernet, Wi-Fi 5/6/6E                           | Internet instável, Wi-Fi 6 |
| Bluetooth             | Versões 4.2, 5.0, 5.2                            | Adicionar Bluetooth, múltiplos dispositivos |
| Water Cooler          | Tipos, diferença para cooler a ar                | Vale a pena, gabinete especial |
| Backup                | Serviços de backup e recuperação                 | Perda de arquivos, backup automático |
| Fontes de Alimentação | Potência, modularidade, confiabilidade           | Fonte ideal para GPU, modular |
| Monitores             | Full HD, Quad HD, 4K                             | Diferença resoluções, GPU especial |
| Multi-Monitores       | Configuração de várias telas                     | Notebook com 2 telas, 3 monitores |
| Velocidade PCs        | CPU, RAM, SSD, GPU, refrigeração                 | Notebook lento, desktop mais rápido |
| Gabinetes             | ATX, Micro-ATX, Mini-ITX, gamer                  | Espaço para GPU, Water Cooler |
| Placa-Mãe             | Tipos, compatibilidade, chipsets                 | PC sem placa-mãe, compatibilidade CPU |
| Periféricos           | Teclados, mouses, headsets                       | Mecânico vs comum, headset gamer |
| Impressoras           | Multifuncionais, jato de tinta vs laser          | Custo por página, toner original |
| Nobreaks              | Nobreaks e estabilizadores                       | Preciso de nobreak, autonomia |
| Licenças              | Windows, Office, antivírus                       | Licença original, antivírus recomendado |
| Redes                 | Roteadores, repetidores, Mesh Wi-Fi              | Melhorar sinal, vale a pena Mesh |
| Produtividade         | Docking, suportes, cadeiras ergonômicas          | Docking serve para quê, conforto |
