# Multi-Monitores

## Dispositivos necessários
- Placas de vídeo com múltiplas saídas
- Docking stations
- Hubs HDMI/USB-C

## Perguntas comuns
- Posso usar dois monitores no meu notebook?
- Qual aparelho permite conectar três monitores?
- Monitores diferentes funcionam juntos?

## Respostas sugeridas
- "Sim, com saída HDMI ou USB-C podemos instalar docking station."
- "Para três monitores, recomendamos placa dedicada ou hub USB-C."
- "É possível usar monitores diferentes, mas resoluções semelhantes melhoram a experiência."
